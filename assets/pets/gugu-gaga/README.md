# gugu-gaga

This folder contains the actual Windows-side pet prototype assets for "得됀눗藤뀐땙".

## Files

- `pet.json`: prototype metadata
- `spritesheet.webp`: character spritesheet

## Metadata

- ID: `gugu-gaga`
- Display name: `得됀눗藤뀐땙`
- Description: A cheerful penguin-suit chibi companion with black bob hair, bright eyes, yellow beak hood, flipper arms, white belly, and yellow feet.
