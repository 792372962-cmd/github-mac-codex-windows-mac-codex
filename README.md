This repository is a small handoff package for Codex across Windows and Mac.

Read [`MAC_Codex_Handoff.md`](./MAC_Codex_Handoff.md) first, then fill in [`PROJECT_CONTEXT_TEMPLATE.md`](./PROJECT_CONTEXT_TEMPLATE.md).

- Preserve the current working context for the "拆解项目"
- Capture the Windows-side "咕咕嘎嘎" pet prototype so Mac Codex can continue from the same source of truth

- Provide a concrete template for the missing project details
